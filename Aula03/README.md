# Control_Systems

Este repositório contém exemplos de manipulação de sistemas físicos com controle utilizando o Octave. O objetivo é analisar sistemas de controle representados por funções de transferência e estudar suas respostas a diferentes entradas.

## Exemplos de Sistemas de Controle

### Exemplo 1: Análise de um Sistema de Controle

Neste exemplo, é realizada a análise de um sistema de controle utilizando uma função de transferência representada por:

$$
g(s) = \frac{100}{s^2 + 2s + 25}
$$

A resposta ao impulso, resposta ao degrau e outros parâmetros do sistema são calculados e analisados.  
**[Veja o código completo aqui](https://github.com/hqnicolas/Control_Systems/blob/main/Aula03/exemplo01)**

### Exemplo 2: Comparação de Duas Funções de Transferência

Neste exemplo, duas funções de transferência diferentes são analisadas e comparadas. As funções são dadas por:

1. $$ g_1(s) = \frac{200}{s^2 + 200s} $$
2. $$ g_2(s) = \frac{200}{s^2 + 200s + 200} $$

A resposta ao degrau de ambos os sistemas é calculada e comparada para entender seu comportamento dinâmico.  
**[Veja o código completo aqui](https://github.com/hqnicolas/Control_Systems/tree/main/Aula03/exemplo02)**
